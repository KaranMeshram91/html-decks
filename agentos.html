Every task, same 6-step pipeline

1
Intake
Parse intent · validate auth · assign task ID
2
Route
Embedding similarity + capability manifest → best-fit agent
3
Context Load
Redis short-term + pgvector chunks + tool manifest + org policies
4
Execute
LLM → tool calls in gVisor sandbox → loop until done
5
Persist & Respond
Write to long-term memory · emit logs · update cost ledger · return result
